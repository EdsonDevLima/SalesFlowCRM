# Atlas CRM Frontend

Aplicação frontend do Atlas CRM construída com React, TypeScript e Vite.

## Comandos principais

```bash
npm install
npm run dev
npm run build
npm run preview
npm run lint
```

## Variáveis de ambiente

O frontend passa a usar o `.env` da raiz do monólito. Ajuste o arquivo `../.env` com:

```env
VITE_API_URL=http://localhost:3000
VITE_SERVER_API_KEY=sua_api_key_interna
```
