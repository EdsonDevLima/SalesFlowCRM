# Atlas CRM API

API REST do Atlas CRM construída com NestJS, TypeORM e MySQL.

## Comandos principais

```bash
npm install
npm run start
npm run start:dev
npm run start:debug
npm run build
npm run start:prod
npm run lint
npm run format
```

## Variáveis de ambiente

Ajuste o arquivo `.env`:

```env
PORT=3000
DATABASE_URL=mysql://usuario:senha@localhost:3306/salesflow
SECRET_JWT=sua_chave_jwt
```
